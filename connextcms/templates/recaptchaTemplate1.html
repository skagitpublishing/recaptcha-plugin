Plugin Example

This is an example plugin for ConnextCMS and KeystoneJS. This screen is an example of a Backbone View. The left menu is also a separate Backbone View. This plugin appended the menu item to the Left Menu View so that you could navigate to this View screen.

View the GitHub repository for this plugin.

The user interface below is an example of manipulating database data through ConnextCMS. You can type any string into the bottom input text box and click the Add button to add the string to the database model. You can also update or delete any existing string in the database by clicking on the respective buttons.
